# Signals

This module extends [Signalization](./Signalization.md) to define some types of `Signal`.

## Dependencies

This module depends on [Signalization](./Signalization.md).

## Architecture

No architecture because this module defines some types of `Signal`.

## Implementation

This module defines

- `Touch`
- `Push`
- `Pull`
- `Tilt`
- `Position`
- `Rotation`
- `Direction`
- `Angle`

as `Signal`.
